# web-euroods-guide

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面，内容涵盖欧洲相关的生活、旅行、文化、政策等方向的参考性资料。每个页面均为独立的静态 HTML 文件，不依赖后端服务，可直接在浏览器中打开访问。

## 目录说明

```
web-euroods-guide/
├── index.html          # 主页面索引
├── pages/              # 存放各个独立 HTML 页面
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
├── assets/             # 静态资源（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

## 页面归档说明

- 每个 HTML 页面均为自包含的完整文档，可直接在浏览器中打开使用。
- 页面内容以信息整理、参考指南为主，不涉及具体商业服务或推广。
- 所有页面均遵循基本的前端开发规范，兼容主流浏览器。

## 维护说明

- 本仓库由维护者不定期更新，新增或修正页面内容。
- 欢迎通过 Issue 提交建议或报告问题，但请注意不要包含具体域名或营销内容。
- 不接受直接向仓库推送未经审核的页面修改请求，如有内容贡献意向，请先通过 Issue 沟通。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用浏览器打开 `index.html` 查看页面列表。
3. 点击列表中的链接可跳转到对应的独立页面。

本仓库内容仅供个人学习与参考使用，不构成任何形式的建议或承诺。
